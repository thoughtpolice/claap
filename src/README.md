File system layout:

```
./src/                <- You are here.
./src/aap/            <- AAP processor, implemented in CLaSH.
./src/clash-extras/   <- Extra code for hooking up CLaSH with Lattice iCE40 boards.
./src/dhrystone/      <- Dhrystone benchmark.
./src/fpga/           <- Extra files needed for particular synthesis flows or boards.
./src/mk/             <- Build system, written with Shake.
./src/t/              <- Test suite.
```
